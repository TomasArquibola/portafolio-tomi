HTML:
El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos. Utilizo este lenguaje para crear la estructura de mis sitios web, como este portafolio.
70%
CSS:
CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML. Utilizo este lenguaje porque me permite darle estilos a mis sitios web, como este portafolio, que tienen colores y diseño gracias a CSS.
60%
Python:
Python es un lenguaje de programación ampliamente utilizado en aplicaciones web, desarrollo de software, ciencia de datos y machine learning (ML). Los desarrolladores utilizan Python porque es eficiente y fácil de aprender, además de que se puede ejecutar en muchas plataformas diferentes. Utilizo este lenguaje por su sintaxis sencilla, lo que me permite entender mejor y aprender a programar.
50%
C++:
C++ proporciona bibliotecas estándar altamente optimizadas y permite el acceso a características de hardware de bajo nivel para maximizar la velocidad y minimizar los requisitos de memoria. Principalmente utilizo este lenguaje para desarrollar programas básicos como calculadoras, cajeros automáticos, gestión de alumnos y cálculos matemáticos.
40%
